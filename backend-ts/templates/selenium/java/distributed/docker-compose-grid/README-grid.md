## Quick Start

```bash
# Start the Grid (hub + Chrome + Firefox nodes)
docker compose up -d selenium-hub chrome-node firefox-node

# Wait for Grid, then run tests
./infra/scripts/run-grid-tests.sh

# Or run everything in one shot
docker compose --profile tests up --abort-on-container-exit

# Tear down
docker compose down
```
