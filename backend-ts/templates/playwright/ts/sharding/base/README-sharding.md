```bash
# Shard 1 of {{TOTAL_SHARDS}}
npx playwright test --shard=1/{{TOTAL_SHARDS}}

# Shard 2 of {{TOTAL_SHARDS}}
npx playwright test --shard=2/{{TOTAL_SHARDS}}

# Merge blob reports
npx playwright merge-reports --reporter html ./blob-report
```

## CI Integration

Copy the job definitions from `infra/sharding/shard-matrix.yml` into your
`.github/workflows/playwright.yml`. The matrix strategy runs all shards in
parallel and auto-merges the final HTML report.
