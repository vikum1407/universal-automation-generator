## Prerequisites

1. [Sign up for BrowserStack](https://www.browserstack.com/users/sign_up)
2. Get your credentials from [Account Settings](https://www.browserstack.com/accounts/settings)
3. Export as environment variables (or add to CI secrets):

```bash
export BROWSERSTACK_USERNAME="your-username"
export BROWSERSTACK_ACCESS_KEY="your-access-key"
```

## Run Tests

```bash
# Selenium/Java
mvn test -DBROWSERSTACK_USERNAME=$BROWSERSTACK_USERNAME \
         -DBROWSERSTACK_ACCESS_KEY=$BROWSERSTACK_ACCESS_KEY

# Playwright/TypeScript
npx browserstack-node-sdk npx playwright test

# Python
browserstack-sdk pytest tests/
```

## Configuration

Edit `infra/cloud/browserstack.conf.json` to adjust:

- `platforms` — browsers and OS combinations
- `parallelsPerPlatform` — concurrent sessions per platform
- `buildName` — visible in BrowserStack Automate dashboard
